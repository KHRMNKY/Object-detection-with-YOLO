## Installation


Projeyi klonlayın

```bash
  git clone https://github.com/KHRMNKY/Object-detection-with-YOLO
```

Terminalden Proje dizinine gidin

```bash
  cd Cattle_Detection_with_YOLOV8
```

Gerekli paketleri yükleyin

```bash
  pip install -r requirements.txt
```
